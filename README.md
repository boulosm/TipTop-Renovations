# TipTop Renovations Website

## Included
- Responsive home page
- Service pages for residential, commercial, landscaping, decks, snow removal and nursery murals
- External link to https://tiptopawnings.ca
- FAQ, gallery, service areas and contact pages
- Shared CSS and JavaScript
- SEO titles, descriptions, canonical links, robots.txt and sitemap.xml
- LocalBusiness / GeneralContractor schema on the home page

## Before publishing
1. Replace stock gallery images with your own project photos.
2. Replace the review placeholders with verified customer reviews.
3. Connect the contact form to Formspree, Netlify Forms, your hosting platform or a server-side form handler.
4. Upload the contents of this folder to the root of the tiptoprenos.ca hosting account.
5. Confirm that the awning domain is exactly `tiptopawnings.ca`.
